# AI-Anime-Recommender-System-
This system will suggest the anime according to the prompt you entered 

Tech Stack: 
1. groq - LLM 
2. HuggingFace - Embedding Model
3. GCP VM - Virtual machine that can be accesed on cloud, its a service offiered by google cloud
4. Longchain - generative AI framework to intract LLM
5. minikube - for making kubernetes cluster where you deploy your application
6. streamlit - to make ui or frontend of the app
7. docker - for containerization of the app during deployement
8. grafana cloud - monitoring the kubernetes cloud
9. chroma DB - local vector store for storing embeddings
10. kubectl - command line interface to interct with your kubernetes
11. github - it will work as a scm for project
